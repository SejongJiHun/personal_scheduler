🧱 추천 주제 리스트
1. 개인화 일정 관리 서비스 (캘린더 + 할 일 관리)
기능: 회원가입, 로그인, 일정 생성/수정/삭제, 알림 설정

특징: REST API 설계, JWT 인증, DB 연동, 스케줄러 (Spring @Scheduled)

임팩트 포인트: CRUD + 인증/인가 + 스케줄링 전부 다루게 된다.

추가 아이디어: Google Calendar API 연동해서 "외부 캘린더 싱크"까지 하면 금상첨화.

2. 실시간 게시판/커뮤니티 서비스
기능: 글 작성/수정/삭제, 댓글, 좋아요, 신고 기능

특징: WebSocket으로 실시간 알림 넣을 수 있음

임팩트 포인트: **API 설계, DB 설계, 실시간 통신(WebSocket)**까지 챙길 수 있음.

추가 아이디어: 간단한 관리자 페이지 (관리자 계정 만들어서 신고글 삭제)도 붙이면 강함.

3. 간단한 이커머스(쇼핑몰) 백엔드 시스템
기능: 상품 등록/조회/수정/삭제, 장바구니, 주문 결제

특징: JPA 복합 키 처리, 결제 로직, 재고 관리

임팩트 포인트: 관계형 데이터베이스 설계, 트랜잭션 처리 연습 가능.

추가 아이디어: 간단한 쿠폰/포인트 시스템 붙이면 난이도 급상승 (가산점)

4. 날씨 기반 추천 서비스 (ex: 오늘 날씨에 맞는 영화 추천)
기능: 위치 기반 날씨 조회, 날씨에 따라 추천 아이템 보여주기

특징: 외부 API 호출 + 캐싱(Redis 써서 날씨 데이터 임시 저장)

임팩트 포인트: 외부 API 사용 경험, 데이터 캐싱, 비즈니스 로직 설계

5. 도서 관리 시스템 (도서관 대여 서비스)
기능: 회원 가입/로그인, 책 등록/대여/반납, 대여 이력 관리

특징: REST API + DB 트랜잭션

임팩트 포인트: 상태 관리(대여중/대여가능), 트랜잭션 관리 제대로 연습 가능

네가 어떤 주제를 고르든, 반드시:

✅ Spring Boot 기반으로
✅ MySQL DB 설계 직접
✅ JPA/QueryDSL 써서 API 작성
✅ Swagger로 API 문서화
✅ GitHub에 코드 + README 작성
✅ AWS EC2에 배포해서 실서비스 가능 상태로

이걸 완성해야 면접에서도 "진짜 실무 가능한 신입"으로 취급받는다.
